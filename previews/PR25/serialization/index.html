Saving results

In just four lines of code, we can setup serialization of collections of PackageAnalyzer's Package object to Apache arrow tables:

julia> using Arrow, PackageAnalyzer

julia> Arrow.ArrowTypes.registertype!(PackageAnalyzer.Package, PackageAnalyzer.Package)

julia> save(path, packages) = Arrow.write(path, (; packages))
save (generic function with 1 method)

julia> load(path) = copy(Arrow.Table(path).packages)
load (generic function with 1 method)

Then we can do e.g.

julia> results = analyze(find_packages("DataFrames", "Flux"));

julia> save("packages.arrow", results)
"packages.arrow"

julia> roundtripped_results = load("packages.arrow")

Note that even if future versions of PackageAnalyzer change the layout of Package's and you forget the version used to serialize the results, you can use the same load function without calling registertype! in the Julia session in order to deserialize the results back as NamedTuple's (instead of as Packages), providing some amount of robustness.
