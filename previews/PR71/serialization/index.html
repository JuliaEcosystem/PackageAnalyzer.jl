Saving results

Since a Vector{Package} is a Tables.jl-compatible row table, one does not need to do anything special to save the results as a table. For example,

julia> using DataFrames, Arrow, PackageAnalyzer
julia> results = analyze(find_packages("DataFrames", "Flux"));
julia> Arrow.write("packages.arrow", results)"packages.arrow"
julia> roundtripped_results = DataFrame(Arrow.Table("packages.arrow"))2×22 DataFrame
